# Weather-Forecast
Weather application developed by a team of AGES students using HTML, CSS, and JavaScript, with integration of the Open-Meteo and Nominatim APIs for weather data and geolocation services.

Desenvolvido por um grupo de estudantes da AGES em HTML, CSS e JavaScript, utilizando APIs como Open-Meteo e Nominatim.
